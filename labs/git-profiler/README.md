## ✨ Features

- **🏆 Developer Ranking**: Get officially ranked from C-Class to S-Tier.
- **📉 Velocity Tracking**: See your coding speed across time.
- **🗣️ Vocabulary Cloud**: What words define your work? (refactor, fix, oops...)
- **🔥 Streak Detection**: How long can you stay in the flow?
- **🎨 Neo-Terminal UI**: ASCII art and neon-infused color schemes.
- **📊 HTML Export**: Generate a beautiful interactive DNA report.

## 🚀 Installation

### Fast Install
```bash
git clone https://github.com/KippieG/git-personality-profiler.git
cd git-personality-profiler
python3 profiler.py
```

### Run on any project
```bash
python3 profiler.py /path/to/your/project
```

## 💡 How It Works (The Science)

1. **Extraction**: We pull every bit of metadata from your `.git` folder.
2. **Analysis**: Our engine calculates temporal patterns and linguistic sentiment.
3. **Sequencing**: Your "Developer DNA" is mapped across 4 core axes.
4. **Ranking**: You are assigned a rank based on volume, impact, and cleanliness.
